SVG プレゼンテーション属性

属性

alignment-baseline

これは、オブジェクトが親に対してフォントのベースラインに沿ってどのように配置されるかを指定するものです。 値: auto|baseline|before-edge|text-before-edge|middle|central|after-edge|text-after-edge|ideographic|alphabetic|hanging|mathematical|inherit、アニメーション: 可

baseline-shift (en-US)

これにより、親テキストコンテンツ要素のドミナントベースラインに対するドミナントベースラインの相対的な位置を変更することができます。 値: auto|baseline|super|sub|<percentage>|<length>|inherit、アニメーション: 可

clip (en-US)

要素のどの部分が見えるかを定義します。 値: auto|shape|inherit、アニメーション: 可

clip-path (en-US)

適用された要素を指定された <clipPath> 要素に結びつけます。 値: none|<FuncIRI>|inherit、アニメーション: 可

clip-rule (en-US)

<clipPath> が対象を切り取る方法を知るために、パスのどの辺がシェイプの内側にあるかを判断する方法を指定します。 値: nonezero|evenodd|inherit、アニメーション: 可

color

間接的な値の候補 (currentcolor) を fill, stroke, stop-color, flood-color, lighting-color の各属性に与えるために使用します。 値: <color>|inherit、アニメーション: 可

color-interpolation (en-US)

グラデーションの補間、色のアニメーション、アルファ合成で使用する色空間を指定します。 値: auto|sRGB|linearRGB|inherit、アニメーション: 可

color-interpolation-filters (en-US)

フィルター効果を通して行われる画像演算で使用する色空間を指定します。 値: auto|sRGB|linearRGB|inherit、アニメーション: 可

color-profile (en-US)

<image> 要素を通して取り込まれるラスター画像に使用すべきカラープロファイルを定義します。 値: auto|sRGB|linearRGB|<name>|<IRI>|inherit、アニメーション: 可

color-rendering

これは、色の補間や合成の操作を最適化するためのヒントをブラウザーに与えます。 値: auto|optimizeSpeed|optimizeQuality|inherit、アニメーション: 可

cursor (en-US)

マウスポインターが要素上に来た場合に表示されるマウスカーソルを指定します。 値: <FuncIRI>|<keywords>|inherit、アニメーション: 可

d

描画するパスを定義します。 値: path() (en-US)|none

direction (en-US)

テキストの基本的な書字方向を指定します。 値: ltr|rtl|inherit、アニメーション: 可

display (en-US)

グラフィックやコンテナー要素の表示を制御することができます。 値: CSS のdisplay、アニメーション: 可

dominant-baseline (en-US)

ボックスのテキストやインラインレベルのコンテンツを配置するために使用されるベースラインを定義します。 値: auto|text-bottom|alphabetic|ideographic|middle|central| mathematical|hanging|text-top、アニメーション: 可

enable-background (en-US)

背景画像の蓄積をどのように管理するかをブラウザーに伝えます。 値: accumulate|new|inherit; アニメーション: 不可

fill

適用されるグラフィック要素の内側の色を定義します。 値: <paint>、アニメーション: 可

fill-opacity

現在のオブジェクトが塗りつぶされる色やコンテンツの不透明度を指定します。 値: <number>|<percentage>、アニメーション: 可

fill-rule (en-US)

パスのどの辺がシェイプの内側にあるかを判断する方法を示します。 値: nonzero|evenodd|inherit、アニメーション: 可

filter (en-US)

<filter> (en-US) 要素で定義されたフィルター効果を、その要素に適用することを定義します。 値: <FuncIRI>|none|inherit、アニメーション: 可

flood-color (en-US)

<feFlood> (en-US) または <feDropShadow> 要素で定義された現在のフィルタープリミティブの部分領域を塗りつぶすために使用する色を示します。 値: <color>、アニメーション: 可

flood-opacity (en-US)

<feFlood> (en-US) または <feDropShadow> 要素で定義された現在のフィルタープリミティブの部分領域で使用する不透明度の値を示します。 値: <number>|<percentage>、アニメーション: 可

font-family (en-US)

要素のテキストを描画するためにどのフォントファミリーが使用されるかを示します。 値: CSS のfont-family を参照、アニメーション: 可

font-size (en-US)

フォントの大きさを指定します。 値: CSS のfont-size を参照、アニメーション: 可

font-size-adjust (en-US)

大文字の高さではなく、小文字の高さに基づいてフォントの大きさを選択することを指定します。 値: <number>|none|inherit、アニメーション: 可

font-stretch (en-US)

フォントのフェイスを normal, condensed, expanded から選択します。 値: CSS の font-stretch を参照、アニメーション: 可

font-style (en-US)

これは、フォントの font-family (en-US) から、通常、イタリック体、斜体のいずれのスタイルにするかを指定します。 値: normal|italic|oblique、アニメーション: 可

font-variant (en-US)

スモールキャップや合字など、フォントのバリエーションの一部を使用するかどうかを指定します。 値: CSS のfont-variant、アニメーション: 可

font-weight (en-US)

フォントの太さを指定します。 値: normal|bold|lighter|bolder|100|200|300|400|500|600|700|800|900、アニメーション: 可

glyph-orientation-horizontal (en-US)

inline-progression-direction が horizontal の場合の字体の向きを制御します。 値: <angle>|inherit; アニメーション: 不可

glyph-orientation-vertical (en-US)

inline-progression-direction が vertical の場合の字体の向きを制御します。 値: auto|<angle>|inherit; アニメーション: 不可

image-rendering (en-US)

画像処理を行う際に、速度と品質のどちらを重視するかのヒントをブラウザーに与えます。 値: auto|optimizeQuality|optimizeSpeed、アニメーション: 可

kerning (en-US)

ブラウザーが字間を調整するかどうかを示します。 値: auto|<length>|inherit、アニメーション: 可

letter-spacing (en-US)

テキスト文字の間の字間を制御します。 値: normal|<length>|inherit、アニメーション: 可

lighting-color (en-US)

フィルタープリミティブ要素 <feDiffuseLighting> (en-US) および <feSpecularLighting> (en-US) のための光源のの色を定義します。 値: <color>、アニメーション: 可

marker-end (en-US)

これは、与えられた <path> 要素や基本図形の最後の頂点に描かれる矢尻やポリマーカーを定義します。 値: <FuncIRI>|none|inherit、アニメーション: 可

marker-mid

これは、与えられた <path> 要素や基本図形の最初と最後を除いた頂点に描かれる矢尻やポリマーカーを定義します。 値: <FuncIRI>|none|inherit、アニメーション: 可

marker-start (en-US)

これは、与えられた <path> 要素や基本図形の最初の頂点に描かれる矢尻やポリマーカーを定義します。 値: <FuncIRI>|none|inherit、アニメーション: 可

mask (en-US)

特定の位置で画像をマスクしたり切り取ったりすることで、要素の可視性を変更します。 値: CSS のmask、アニメーション: 可

opacity (en-US)

単一のオブジェクトまたはオブジェクトのグループの透過性を指定します。 値: <opacity-value>、アニメーション: 可

overflow (en-US)

ブロックレベル要素のコンテンツが要素のボックスからはみ出したときに切り取るかどうかを指定します。 値: visible|hidden|scroll|auto|inherit、アニメーション: 可

pointer-events (en-US)

要素がマウスイベントのターゲットになれるかどうか、いつなるかを定義します。 値: bounding-box|visiblePainted|visibleFil|visibleStroke|visible |painted|fill|stroke|all|none、アニメーション: 可

shape-rendering (en-US)

ブラウザーが <path> 要素や基本的な図形を描画する際に、どのようなトレードオフを行うかについてのヒントです。 値: auto|optimizeSpeed|crispEdges|geometricPrecision |inherit、アニメーション: 可

solid-color

- 値:; Animatable: -

solid-opacity

- 値:; Animatable: -

stop-color

グラデーションの色停止点で使用する色を示します。 値: currentcolor|<color>|<icccolor>|inherit、アニメーション: 可

stop-opacity (en-US)

グラデーションの色停止点の(不)透明度を定義します。 値: <opacity-value>|inherit、アニメーション: 可

stroke

図形の輪郭を描くのに使用する色を定義します。 値: <paint>、アニメーション: 可

stroke-dasharray

図形の輪郭を描くのに使用する破線と隙間のパターンを定義します。 値: none|<dasharray>、アニメーション: 可

stroke-dashoffset

関連するダッシュ配列を描画する際のオフセットを定義します。 値: <percentage>|<length>、アニメーション: 可

stroke-linecap

開いたサブパスを描いたときに、その端点に使用する形状を定義します。 値: butt|round|square、アニメーション: 可

stroke-linejoin

パスの角を描いたときに使用する形状を定義します。 値: arcs|bevel|miter|miter-clip|round、アニメーション: 可

stroke-miterlimit

マイター結合の描画に使用される stroke-width (en-US) に対するマイターの長さの比率の制限を定義します。 値: <number>、アニメーション: 可

stroke-opacity (en-US)

図形の線の(不)透明度を定義します。 値: <opacity-value>|<percentage>、アニメーション: 可

stroke-width (en-US)

図形に適用される線の太さを定義します。 値: <length>|<percentage>、アニメーション: 可

text-anchor (en-US)

テキストの文字列の垂直方向の配置を定義します。 値: start|middle|end|inherit、アニメーション: 可

text-decoration (en-US)

テキストの装飾線の外見を設定します。 値: none|underline|overline|line-through|blink|inherit、アニメーション: 可

text-rendering (en-US)

ブラウザーでテキストを描画する際に何をトレードオフにするかについてのヒントです。 値: auto|optimizeSpeed|optimizeLegibility|geometricPrecision|inherit、アニメーション: 可

transform

要素と要素の子に適用される座標変換定義のリストを定義します。 値: <transform-list>、アニメーション: 可

unicode-bidi (en-US)

- 値:; Animatable: -

vector-effect (en-US)

オブジェクトを描画する際に使用するベクター効果を指定します。 値: default|non-scaling-stroke|inherit|<uri>、アニメーション: 可

visibility (en-US)

グラフィック要素の可視性を制御することができます。 値: visible|hidden|collapse|inherit、アニメーション: 可

word-spacing (en-US)

単語間の空白の動作を指定します。 値: <length>|inherit、アニメーション: 可

writing-mode

<text> 要素の最初のインライン進行方向を、左から右、右から左、上から下のどれにするかを指定します。 値: lr-tb|rl-tb|tb-rl|lr|rl|tb|inherit、アニメーション: 可

Firefox 89 for developers

このページでは、開発者に影響する Firefox 89 の変更点をまとめています。Firefox 89 は、米国時間 2021 年 6 月 1 日にリリースされました。

Note: Mozilla Hacks の Looking fine with Firefox 89 もご覧ください。

ウェブ開発者向けの変更点一覧

開発者ツール

変更なし。

HTML

変更なし。

CSS

  • forced-colors メディア特性を実装しました (バグ 1659511)。
  • @font-face の ascent-override、descent-override、line-gap-override ディスクリプターを実装しました (バグ 1681691 および バグ 1704494)。
  • image-set() の type() 関数を実装しました (バグ 1695404)。

JavaScript

  • トップレベルの await をデフォルトで有効にしました (バグ 1681046)。
  • 64 ビットシステムで、長さが 2GB-1 より大きい (最大 8GB) ArrayBuffer を作成できるようになりました (バグ 1703505)。

HTTP

変更なし。

API

DOM

  • PerformanceEventTiming (en-US) をデフォルトで有効にしました (バグ 1701029)。
  • <input> および <textarea> 要素の内容物を、デフォルトで Document.execCommand() コマンドを使用して操作できるようになりました。contentEditable などの冗長な回避策なしに、編集履歴の維持やほかのブラウザーとの同等性を提供します (バグ 1220696)。

廃止

  • 以下のセンサーイベントおよび関連するハンドラーを削除しました (主にほかの主要なブラウザーエンジンとの互換性を向上するため、またプライバシー侵害の懸念に対処するため):
    • DeviceProximityEvent および window.ondeviceproximity イベントハンドラー (バグ 1699707)。
    • UserProximityEvent および window.onuserproximity) イベントハンドラー (バグ 1699707)。
    • DeviceLightEvent および window.ondevicelight イベントハンドラー (バグ 1701824)。

WebDriver conformance (Marionette)

廃止

  • WebDriver 仕様書に含まれていない rotatable 能力を使用できなくなりました (バグ 1697630)。

アドオン開発者向けの変更点

  • 動的な JS モジュールの読み込み が、WebExtension のコンテンツスクリプトで動作するようになりました (バグ 1536094)。
  • web_accessible_resources に記載した拡張機能のリソースが、要求の CORS モードにかかわらず読み込まれるようになりました (バグ 1694679)。
  • Firefox の UI を再設計したため、theme API の使用に影響があります。tab_background_separator および toolbar_field_separator プロパティをサポートしなくなりました。tab_line および toolbar_vertical_separator の動作が変わりました。詳しくは Changes to themeable areas of Firefox in version 89 をご覧ください。
  • pageAction ボタンをアドレスバーにピン止めおよびピン止め解除することができなくなりました。デフォルトで三点リーダーアイコンのメニューが表示されないためです (バグ 1691454)。この結果、page_action マニフェストキーの pinned プロパティは効果がなくなりました (バグ 1703537)。
  • pageAction ボタンから、コンテキストメニューの "アドレスバーから削除" 項目を削除しました (バグ 1704474)。この機能の代替手段について バグ 1712556 をご覧ください。

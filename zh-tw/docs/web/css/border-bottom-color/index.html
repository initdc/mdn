border-bottom-color

border-bottom-color 是一種CSS屬性，用來設定元件下側邊框的顏色。也可由含括此屬性功能的簡寫屬性border-color或border-bottom (en-US)設定該屬性值。

語法

/* <color> values */
border-bottom-color: red;
border-bottom-color: #ffbb00;
border-bottom-color: rgb(255, 0, 0);
border-bottom-color: hsla(100%, 50%, 25%, 0.75);
border-bottom-color: currentcolor;
border-bottom-color: transparent;

/* Global values */
border-bottom-color: inherit;
border-bottom-color: initial;
border-bottom-color: revert;
border-bottom-color: revert-layer;
border-bottom-color: unset;

border-bottom-color只能宣告一個屬性值。

屬性值

<color> (en-US)

定義下側邊框色彩。

Formal definition 標準定義

預設值currentcolor
Applies toall elements. It also applies to ::first-letter.
繼承與否no
Computed value (en-US)computed color
Animation typea color

Formal syntax 語法規則

<'border-top-color'>

範例

簡易邊框範例

HTML

<div class="mybox">
  <p>這是一個帶有框線的方框。
     請注意某側框線顏色已設為
     <span class="redtext">紅色</span>。</p>
</div>

CSS

.mybox {
    border: solid 0.3em gold;
    border-bottom-color: red;
    width: auto;
}

.redtext {
    color: red;
}

參見

  • 與框線相關的簡寫屬性：border、border-bottom (en-US)、border-color。
  • 與顏色相關的其他框線屬性：border-right-color、border-top-color、border-left-color。
  • 與同側框線相關的其他屬性：border-bottom-style (en-US)、border-bottom-width (en-US)。

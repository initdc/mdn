HTMLBodyElement

HTMLBodyElement интерфейс предоставляющий специальные свойства (помимо этого, они постоянно наследуют HTMLElement интерфейс) для манипуляции элементами тела.

Свойства

Наследует свойства о его родителя, HTMLElement и от WindowEventHandlers.

HTMLBodyElement.aLink
Это DOMString который представляет цвет активных гиперссылок.
HTMLBodyElement.background
Это DOMString который представляет собой описание расположения фонового изображения ресурса. Обратите внимание, что это не URI, хотя в некоторых старых версиях некоторых браузеров ожидается именно это.
HTMLBodyElement.bgColor
Это DOMString который представляет собой цвет фона для документа.
HTMLBodyElement.link
Это DOMString который представляет собой цвет непосещенных ссылок.
HTMLBodyElement.text
Это DOMString который представляет собой основной цвет текста.
HTMLBodyElement.vLink
Это DOMString который представляет собой цвет посещённых ссылок.

Методы

Не имеет специфичных методов; наследует методы от его родителя, HTMLElement и от WindowEventHandlers.

Обработчики событий

Не имеет специфичных обработчиков событий; наследует обработчики событий от его родителя, HTMLElement и от WindowEventHandlers.

WindowEventHandlers.onafterprint
Это event handler представляющий собой код который будет вызван когда afterprint (en-US) событие будет вызвано.
WindowEventHandlers.onbeforeprint (en-US)
Это event handler представляющее собой код который будет вызван когда событие beforeprint (en-US) вызвано.
WindowEventHandlers.onbeforeunload
Это event handler представляющее собой код который будет вызван когда событие beforeunload (en-US) вызвано.
WindowEventHandlers.onhashchange
Это event handler представляющее собой код который будет вызван когда событие hashchange (en-US) вызвано.
WindowEventHandlers.onlanguagechange (en-US)
Это event handler представляющее собой код который будет вызван когда событие languagechange (en-US) вызвано.
WindowEventHandlers.onmessage (en-US)
Это event handler представляющее собой код который будет вызван когда событие message вызвано.
WindowEventHandlers.onoffline
Это event handler представляющее собой код который будет вызван когда событие offline (en-US) вызвано.
WindowEventHandlers.ononline
Это event handler представляющее собой код который будет вызван когда событие online (en-US) вызвано.
WindowEventHandlers.onpagehide
Это event handler представляющее собой код который будет вызван когда событие pagehide (en-US) вызвано.
WindowEventHandlers.onpageshow
Это event handler представляющее собой код который будет вызван когда событие pageshow (en-US) вызвано.
WindowEventHandlers.onpopstate
Это event handler представляющее собой код который будет вызван когда событие popstate (en-US) вызвано.
WindowEventHandlers.onresize
Это event handler представляющее собой код который будет вызван когда событие resize (en-US) вызвано.
WindowEventHandlers.onstorage (en-US)
Это event handler представляющее собой код который будет вызван когда событие storage (en-US) вызвано.
WindowEventHandlers.onunload
Это event handler представляющее собой код который будет вызван когда событие unload (en-US) вызвано.
